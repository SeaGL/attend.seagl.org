## Bot

Here are the commands you can use with the bot!

- `!info`: Provide URL to SeaGL info page
- `!help`: Provide URL to SeaGL-bot user guide
- `!ask <question>`: Accept a question for a speaker.
- `!tea <user-id>`: Send a fellow user a toast.
- `!create-room <room-name>`: Create a new chat-room and provide a URL which will link to a new video chat room.
