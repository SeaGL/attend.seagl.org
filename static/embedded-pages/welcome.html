Welcome to SeaGL 2021

Our virtual conference space exists in Matrix, an open, decentralized chat network. This website is a Matrix client that integrates features of the conference.

Getting Started

Matrix accounts are decentralized similarly to email—your account is “at” the provider of your choice, but you can communicate seamlessly across providers. To begin:

  • If you already have a Matrix account at any provider, you can sign in with it below.
  • If you’d just like to participate in the conference and aren’t interested in using Matrix further, you can create a temporary account with SeaGL. The account will be deleted at the end of the conference.
  • If you’d like to have a Matrix account that can be used outside of the conference, we recommend creating an account at Matrix.org.
